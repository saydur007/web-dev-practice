So here I used the angular framework to create a simple web page which has a form and slight animation on it.

How I installed the frame work?

I installed it by putting a script tag in the head of my html file. So I added one script tag for installation of angular and another one for angular animation.

How the page was built

The web page was built with a mix of html and angular js. So we used angular js to get the form scope and use it to display all the input. So I set an initial scope in my script tag but whenever someone updates the form the scope changes based on input and is displayed simultaneously. I also added a hide option to hide the form which uses angular animation. SO basically when the checkbox is ticked it instantly trigger the animation hide option of angular and hides the form and if it is unchecked the form reappears with an animation.

Difficulties encountered

So the processing of the form was very straight forward as we just used the input from user to display it simulatenously. Just had to figure out how to use the scope function on angular. the hide animation was a bit tricky as we had to use angular animation on it so we had to create css property for the hide option so that everything vanishes and add a transition speed as well. But angular overall was sufficiently simple and the challenges were overcomed easily
